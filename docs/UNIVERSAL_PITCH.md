## 🎯 任意场景使用流程

1. **读 JD**：把 JD 里出现的技术名词、岗位职责关键词圈出来（一般 3-5 个）
2. **选模板**：根据场景选超短 / 中 / 长版
3. **填关键词**：**用 JD 里的原词**替换 `[方括号]` 内的占位符
4. **检查链接**：保证 GitHub + Demo URL 是 https 完整可点
5. **附简历**：邮件渠道必须附 PDF；Boss 直聘可以晚一点再发
6. **保留截图**：投出去前截一份记录到 Excel / Notion，方便跟进

## 🎁 最后送你 3 句话

1. **被拒不是你不行，是匹配不上**：HR 一天看 200 简历，5 秒决定看不看。**有 Demo 链接的简历秒杀 90%**。

2. **数量 + 质量并行**：一周投 20 家，跟 20 个 HR 聊过。**有 1 个面试就值回票价**。

3. **别等"完美简历"**：你这份简历已经够了。**今天就开始投，下周拿面试**。
